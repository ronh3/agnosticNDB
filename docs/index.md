# agnosticDB Docs

## Overview
agnosticDB is a standalone Mudlet package for tracking Achaea characters via the public API, honors capture, and in-game list ingestion. This folder contains the detailed reference material.

## Guides
- `commands.md`: full command reference.
- `data-schema.md`: DB fields, defaults, and semantics.
- `lua-api.md`: getter helpers and module APIs.
- `rate-limits.md`: API queue/backoff behavior.
- `import-export.md`: JSON formats and default paths.
- `honors.md`: honors parsing behavior and limits.
- `themes.md`: themes and palette usage.
- `integration.md`: Mudlet requirements and dependencies.
